==== Building: ====

$ mkdir build
$ cd build
$ cmake ..
$ make

==== Dependencies: ====

liblo - http://liblo.sourceforge.net

==== Running: ====

$ ./xoscmain 57300
will listen on port 57300 for incoming OSC messages.
